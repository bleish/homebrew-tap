# Bleish Tap

## How do I install these formulae?

`brew install bleish/tap/<formula>`

Or `brew tap bleish/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "bleish/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
